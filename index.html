Mitt blogginlägg

May 1, 2020, Posted by Ludvig

Jag tycker att den här uppgiften var rolig och lärorik. Det var också en skön variation från vår senaste uppgift i C# som var tungt på back-end. Detta blev som en nypa frisk luft med mindre problemlösning och mer design, och som dessutom har varit roligt.
I början när man lär sig CSS så tycker jag att man ofta hanmnar i bråk med språket. Man vet hur man vill ha en design, men att man upplever att CSS inte alltid gör som man vill. Men ju mer tid man lägger på CSS, desto mindre bråkande blir det, tycker jag. Det finns så många olika sätt att lösa sina designutmaningar, så det kan vara svårt att veta vilken lösning som blir bäst. Sedan är det sällan det går på första försöket, att CSS ska göra som man vill. Särskilt om man blandar olika tillvägagångsätt. Men ack så skönt det är i slutet när allting bara fungerar. Ännu skönare när alltid dessutom fungerar precis likadans på olika skärmstorlekar och olika webbläsare. (...)
